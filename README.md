# Research Help

These are a collection of JavaScript files I've written over the years to help with research. 

## Description

Whether it is prompts for generating ideas, or flash cards for memorizing definitions and theorems, or something more detailed like a drag and drop system to help me brainstorm a proof, These are the things that my mind doodles into and helps me to construct my ideas. 

## Getting Started

### Dependencies

* These are all client-side JavaScript dependent. This means they require you to have JS enabled. 

### Installing

* There should not be any instillations required. 
  
### Executing program

* Executions are different for each program and they should come with their own sets of instructions. 
 
```
code blocks for commands
```
